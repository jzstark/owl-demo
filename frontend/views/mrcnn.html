Demo: Image Segmentation with Mask R-CNN

This is an implementation of the Mask R-CNN network using Owl. This network can be used to perform object detection, segmentation and classification. The implementation is based on this paper , and ported from this Keras implementation . The code used in the demo can be found in this Github Repo .

Belows shows one example of applying Mask R-CNN to an image of street view. You can also upload your own image to be processed. Note that to reduce server load, images are resized to a small size before being processed, which diminishes the quality of the detection. You can obtain better results by running the code locally.
